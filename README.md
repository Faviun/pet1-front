# Frontend часть интернет-магазина.
Клиентская часть интернет-магазина по продаже котлов и отопительного оборудования, разработанная на Next.js с использованием TypeScript. Предоставляет пользовательский интерфейс для просмотра каталога, оформления заказов и взаимодействия с backend-API.

Проект создан в учебных целях и представляет собой pet-проект, разрабатываемый по урокам с YouTube.

## Реализованный функционал:

- Регистрация и авторизация пользователей.
- Каталог товаров с фильтрацией по параметрам (цена, характеристики, производитель).
- Карточка товара с детальным описанием и изображениями.
- Поиск по каталогу.
- Корзина покупок и оформление заказов.
- Форма обратной связи с отправкой на email.
- Адаптивный дизайн для мобильных устройств.

## Статус разработки
Проект находится в активной разработке. Интерфейс постоянно улучшается и дополняется новым функционалом по мере развития backend-части.
